# 📱 Teen Smartphone Usage Analysis

An end-to-end data analysis project that explores smartphone usage patterns among teenagers using Python. The project performs data cleaning, exploratory data analysis (EDA), statistical visualization, and extracts meaningful insights about screen time, app usage, sleep habits, academic performance, and smartphone addiction indicators.

---

## 📌 Project Overview

Smartphones have become an integral part of teenagers' daily lives. This project analyzes a teen smartphone usage dataset to identify behavioral patterns and understand how smartphone usage relates to various lifestyle and academic factors.

The analysis focuses on:

- Daily screen time
- Most-used applications
- Sleep duration
- Academic performance
- Social media usage
- Gaming habits
- Smartphone dependency
- Usage trends across different demographic groups

---

## 🎯 Objectives

- Clean and preprocess raw smartphone usage data.
- Perform Exploratory Data Analysis (EDA).
- Visualize usage patterns using informative charts.
- Identify relationships between screen time and other variables.
- Generate actionable insights from the dataset.
